### 关键发现
- 登录页: 用 `input[placeholder*='邮箱']` / `input[placeholder*='密码']`，不用 `#username`
- 智能体按钮点击触发 `widget/track?agentId=XXX` API + 新标签页打开
- 新标签页 URL 格式: `/ai/gui/chat/a_xxx`
- entry_url 格式: `https://agent.digitalchina.com/widget/open?agentId={id}&detail={path}`

2. 点击"打开"按钮访问智能体
3. 对话型智能体 → 调 LLM 随机生成问题 → 测试对话
4. 测试结果写入报告
5. 访问不了的智能体如实报告问题

### 注意事项
1. **必须先登录再操作**: 使用 `login.login(browser_mgr)`
2. **新标签页用 `expect_page()` 拦截**: 不是 `window.open` 拦截
3. **用 `textContent` 而非 `innerText`**: React 虚拟滚动兼容
4. **运行前设置 PATH**: `export PATH="/home/node/.local/bin:$PATH"`
5. **登录页选择器**: 用 `input[placeholder*='邮箱']` 和 `input[placeholder*='密码']`，不用 `#username`/`#password`
