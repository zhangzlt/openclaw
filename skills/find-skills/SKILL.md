# /find-skills — 技能发现

## 触发条件
用户需要新技能、不确定用什么技能、想扩展能力范围。

## 执行流程

### 1. 需求分析
- 明确用户想解决的问题类型
- 匹配关键词到技能领域

### 2. 搜索策略
- 检查本地已安装技能列表
- 使用 skill_workshop list 查看可用提案
- 通过 web_search 查找社区技能

### 3. 推荐输出
- 匹配到的技能列表（名称 + 简介）
- 安装建议和优先级
- 使用示例

### 4. 安装
- 引导用户通过 skill_workshop 安装
- 安装后自动验证可用性
